# SkyWayMultiVoiceChat

### SkyWayを使用した複数人対応の音声チャットです。

音声チャットとテキストチャットが可能です。

以下のURLで見れます。

<https://kadoshita.github.io/SkyWayMultiVoiceChat/>

以下のサンプルコードを使用しました。

<https://github.com/nttcom/SkyWay-MultiParty>

### 動作環境

* Chrome,FireFoxなどのWebRTCに対応したWebブラウザ

### 開発環境

* SublimeText3
* SkyWay
